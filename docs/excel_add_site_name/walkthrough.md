# 修正内容の確認: エクセル出力への現場名追加

エクセル出力の各シートへ現場名を追加する実装が完了しました。

## 変更内容
1. `src/utils/excelExportUtils.js` の `exportToExcel` 関数を修正し、「現場サマリー」「作業項目別詳細」「作業員別集計」の各シートデータの先頭（1行目）に `現場名: 〇〇` を追加しました。
2. ヘッダーのスタイル適用部分を修正し、1行目のセルを列幅に合わせて結合し、太字・左寄せ・中央揃えの見出しとなるようにしました。従来のテーブルヘッダーのスタイル適用は2行目以降に自動でずれるように調整済みです。

## 検証手順（ユーザー側で実施をお願いします）
1. ブラウザでアプリ（管理アプリ / Admin App）を開きます（すでに起動中の場合はそのまま操作可能です）。
2. 「工事設定」タブまたは「ホーム」タブで適当な現場を選択します。
3. 「管理シート」タブを開き、右上の「エクセル出力」ボタン（緑色のボタン）をクリックします。
4. ダウンロードされたExcelファイルを開き、3つのシート全ての1行目に現場名が結合セルとして適切に表示されているか確認してください。
5. 2行目以降のテーブル表示や罫線等のスタイルに崩れがないことも併せてご確認ください。

問題なく出力されていれば、本タスクは完了です！
