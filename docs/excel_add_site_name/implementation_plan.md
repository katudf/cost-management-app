# エクセル出力への現場名追加

エクスポートされるExcelの各シートに、どの現場のデータか分かるよう現場名を追記します。

## 変更内容

### `src/utils/excelExportUtils.js`
各シート（「現場サマリー」「作業項目別詳細」「作業員別集計」）の1行目に現場名が太字で表示されるように修正します。

#### [MODIFY] excelExportUtils.js(file:///c:/Users/katuy/Desktop/cost-management-app/src/utils/excelExportUtils.js)
1. シートデータの先頭に `["現場名: ${activeProject.siteName}"]` という行を追加します。
2. `applyStyleToSheet` 関数に列数を渡すよう引数を追加し、内部で以下の処理を行います。
   - 1行目のセル（A1から末尾列まで）を結合（マージ）する。
   - 1行目のフォントを太字にし、左寄せ・中央揃えのスタイルを適用する。
   - 従来のヘッダー行（インデックス0）に対するスタイル適用を、新しいヘッダー行（インデックス1）へずらす。

## 検証計画
### 手動検証
1. 現在起動中のアプリ（localhost）をブラウザで開きます。
2. Admin Appにて適当な現場を選択し、「管理シート」タブを開きます。
3. エクセル出力ボタンをクリックし、Excelファイルをダウンロードします。
4. ダウンロードしたExcelファイルを開き、3つのすべてのシートの先頭行に「現場名: 〇〇」が表示されていること、またセルの結合・スタイル・表示崩れがないかを目視で検証します。
