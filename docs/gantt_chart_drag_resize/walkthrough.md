# 配置予定表ガントチャート直感的操作の実装完了レポート

ガントチャートのバーをマウス操作で直接伸縮・移動できる機能を実装しました。

## 実装された機能

配置表の上部に表示されている「案件バーチャート（色付きのバー）」に以下の3つの操作アクションが実装されています。

### 1. 開始日の変更（左リサイズ）
バーの「**左端**」付近にマウスカーソルを合わせるとリサイズアイコン（↔）が表示されます。そのままドラッグすることで、**終了日は固定したまま**、開始日だけを伸縮できます。

### 2. 終了日の変更（右リサイズ）
バーの「**右端**」付近にマウスカーソルを合わせるとリサイズアイコン（↔）が表示されます。そのままドラッグすることで、**開始日は固定したまま**、終了日だけを前後に延長・短縮できます。

### 3. 全体工期の移動（中央ドラッグ）
バーの「**両端以外の中央部分**」をドラッグすると、バー全体を左右に移動させることができます。
この機能の最大の特徴は、**「実働日数（休日を除いた工事日数）」が維持される**点です。

**挙動の詳細:**
- 内部で「会社休日設定（ピンク・水色など）」および「カレンダーの土日」を判定しています。
- 例えば「月〜水」の3日間（実働3日）を右にドラッグして金曜日開始にした場合、土日が挟まるため「金・月・火」の範囲にバーが自動的に引き伸ばされ、視覚的なバーの長さはカレンダー上で自動調整されます。

## 使用上のポイント（動作確認）
- 左端・中央・右端でマウスカーソルの表示が異なります。
- ドラッグ操作中はバーが少し半透明になり、動かした先の日付がリアルタイムでプレビューされます。
- マウスのボタンを離した瞬間に、確定された「開始日」「終了日」がデータベース（Projectsテーブル）に保存され、即座にクラウドへ反映されます。
