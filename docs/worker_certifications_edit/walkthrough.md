# 資格情報の追加・編集機能 Walkthrough

## 変更内容の概要
「システム設定」タブ内に、作業員の「資格情報一覧」および「追加・編集用フォーム」を実装しました。これにより、アプリの画面上から直接、資格の登録・更新・削除が可能になりました。

## 実装の詳細
1. **データ連携 (`AdminApp.jsx`)**
   - `SystemSettingsTab` に、全作業員のデータ (`workers`) とデータ再取得関数 (`fetchAllData`) を Props として渡すように修正しました。

2. **管理インターフェースの実装 (`SystemSettingsTab.jsx`)**
   - 既存の「システム共通設定」の下部に、新たに「資格情報マスター」のセクションを追加しました。
   - **フォーム領域**: 作業員名（ドロップダウン）、資格名、取得日、有効期限を入力できるフォームを設置しました。
   - **一覧表示領域**: 登録済みの全作業員の資格情報を統合し、テーブル形式で表示します。行の右端にある「編集」「削除」アイコンで各操作を行います。
   - **CRUD処理**: Supabase の `WorkerCertifications` テーブルとの通信処理を実装。追加・更新・削除の直後に `fetchAllData` を呼び出すことで、画面の再読み込みなしで即時にリストや作業員詳細モーダル側に反映されるようにしています。
