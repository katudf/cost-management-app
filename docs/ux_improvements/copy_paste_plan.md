# 実装計画: 複数セルのコピー・カット・ペーストおよび削除

## Goal
配置表において、マウスで選択した複数日のスケジュールをキーボードショートカット（操作）を使って一括でコピー、カット、ペースト、削除できるようにします。
これにより、スプレッドシートやExcelのような爆速のスケジュール編集体験を提供します。

## 対象となる操作（キーボードショートカット）
配置表で任意のセル（またはドラッグで複数範囲）を選択してポップアップが開いている状態のときに、以下のキーボード操作を可能にします。

1. **Delete / Backspace キー (削除)**
   - 選択中のセル（複数日も可）に配置されているすべてのスケジュール（現場・その他）を一括で削除します。
2. **Ctrl + C (コピー)**
   - 選択中のセルのスケジュールを内部クリップボードにコピーします。
3. **Ctrl + X (カット)**
   - 選択中のセルのスケジュールをクリップボードにコピーし、元の場所からは削除します（移動用）。
4. **Ctrl + V (ペースト)**
   - 選択中のセルを起点として、クリップボードの内容（複数日分のスケジュール）を展開して貼り付けます。

## Proposed Changes

### `AssignmentChartTab.jsx` の改修
*   **クリップボード状態の追加**:
    `const [clipboard, setClipboard] = useState(null)` を追加し、コピーされた配置データ（相対的な日付のズレを含む）を保持します。
*   **キーボードイベントの監視**:
    `useEffect` を用いてグローバルな `keydown` イベントリスナーを追加します。
    現在ポップアップが開いているセルの情報 (`editCell` ステート) を参照し、`Ctrl+C`, `Ctrl+X`, `Ctrl+V`, `Delete` の押下を検知します。
*   **一括削除処理の実装**:
    Supabaseへ `delete` リクエストを配列（`in` フィルタ）を用いて一括で送り、成功時にローカルの `assignments` ステートから一括で取り除きます。
*   **一括配置処理の実装**:
    起点となる日付から `clipboard` の相対日数分ずらした日付を計算し、Supabaseへ `insert` リクエストを一括で送り、ローカルの `assignments` へ反映させます。
*   **視覚的フィードバック**:
    「コピーしました」「X件の配置を削除しました」といったトースト通知(`showToast`)を表示し、操作が成功したことをユーザーに分かりやすく伝えます。
