# 実装計画: 現場名のドラッグ＆ドロップ配置

## Goal
現在のシステムでは「セルをクリック（またはドラッグで複数範囲を選択）して、開いたポップアップから現場を選ぶ」仕組みになっています。
これをさらに直感的にするため、**「画面上部の案件リスト（現場名・工期バー）をマウスで掴み、そのまま作業員のセルへドラッグ＆ドロップして配置する機能」**を追加します。

## Proposed Changes

### 1. `AssignmentChartTab.jsx` へのドラッグ元 (Draggable) の設定
- 画面上部の「案件バーチャート」部分における **現場名セル** に対して、HTML5標準のドラッグ＆ドロップ機能（`draggable={true}` と `onDragStart` イベント）を付与します。
- ドラッグ開始時に、その現場の `projectId` を内部データ（`dataTransfer`）として保持します。

### 2. 作業員セルへのドロップ先 (Drop Zone) の設定
- 作業員が並んでいるスケジュールマス（現在のクリック対象セル）に対して、`onDragOver` と `onDrop` のイベントを追加します。
- `onDragOver` 時にはマウスカーソルが「コピー（追加）」の見た目になり、セルがハイライトするようにします。
- `onDrop` されたとき、掴んでいた `projectId` を読み取り、対象の作業員・対象の日付に対して直接 `addAssignment` を呼び出してデータベースへ即座に配置を保存します。

### 3. 既存の「複数日ドラッグ選択」との干渉防止
- 既存の「マウスを押しっぱなしにして複数日を選択する独自機能」は作業員のマス目に適用されておりHTML5標準ドラッグとは異なるため、直接的な競合は起こりません。しかし、安全のためイベントハンドリング（`e.preventDefault()` などの伝播停止）を慎重に行い、両方の操作性を両立させます。
