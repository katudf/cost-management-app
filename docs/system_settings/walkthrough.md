# システム設定（人工費用動的設定）追加の確認

## 実装内容
ハードコーディングされていた「1人あたりの人工（にんく）費用」（`HOURLY_WAGE`）を廃止し、システム画面上から設定・変更できるようにしました。

### 1. データベースの変更
- `system_settings` テーブルを作成し、`hourly_wage`（人工費用/時）を保存できるようにしました。
- 初期値はご要望通り **3,500円/時** （1人工＝28,000円）に設定されています。

### 2. アプリケーションの変更
- **システム設定画面の追加**: 管理者アプリのサイドメニューに「システム設定」タブを追加しました。ここから1時間あたりの人工費用を自由に変更できます。
- **予測粗利の一括再計算**: システム設定画面で人工費用を変更すると、ダッシュボード上の全現場の「予測粗利」が即座に新しい費用に基づいて再計算されます。
- **Excelインポート時の目標時間計算**: 「工事設定」タブから見積書（Excel）を取り込む際も、システム設定の人工費用（デフォルト3,500円）を基準にして各作業の「目標時間」が自動計算されるように変更しました。

## 確認手順（手動テスト）
1. 管理者アプリのメニューから **「システム設定」** を開きます。
2. 現在の設定値が「3500 円 / 1時間」になっていることを確認します。
3. 値を変更して「設定を保存する」ボタンを押し、保存完了メッセージが表示されることを確認します。
4. **「ホーム」** または **「管理シート」** に戻り、各現場の予測粗利の金額が設定値に応じて変わっていることを確認してください。
5. （任意）ダミーの見積書をExcelインポートし、目標時間が新しい時給で計算されるか確認します。

## 課題・留意事項
- 現在のシステム設定は全現場共通の設定となります。
- 新しくシステム設定テーブルを追加したため、起動時に設定が取得できない場合はデフォルトで 3,500円 が適用されます。
