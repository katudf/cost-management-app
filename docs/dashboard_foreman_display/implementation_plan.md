# ダッシュボード工事一覧の職長名表示 - 実装計画

## 概要
ダッシュボードの工事一覧（登録済み工事一覧）の各プロジェクトカード内に、設定されている「職長名」を表示します。

## 変更内容

### `src/AdminApp.jsx`
- **[MODIFY]** `src/AdminApp.jsx`
  - `lucide-react` から `User` アイコンをインポートに追加します。
  - ダッシュボードのプロジェクトカード部分（`allProjectsSummary.map` 内）のUIに、職長名を表示する要素を追加します。
  - 職長名は、プロジェクトの `foreman_worker_id` を用いて、コンポーネントが保持している `workers` リストから該当するユーザーの名前を検索して表示します。

## 検証計画
- `npm run dev` でローカルサーバーを起動し、ブラウザでアプリケーションにアクセスします。
- 「ホーム」タブ（ダッシュボード）の各工事カードに「職長: 〇〇」という表記が追加されていることを目視確認します。
- 職長が未設定の工事には「未設定」または何も表示されない等の適切なフォールバックが行われることを確認します（今回は「未設定」と表示する想定）。
