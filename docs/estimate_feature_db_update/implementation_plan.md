# 見積書機能に伴うデータベース定義書の更新計画

実施された見積書機能用のデータベースマイグレーションに基づき、`docs/database/database_info.md` を最新化します。

## ユーザーレビューが必要な事項

> [!NOTE]
> **ドキュメントの更新のみです**
> ユーザー様が既に実行されたSQLの内容をドキュメントに正確に反映します。実際のデータベース操作は行いません。

## 提案される変更内容

### 1. 既存テーブルの更新定義の反映
- **Workers**: `stamp_url`（印影画像URL）の追加。
- **system_settings**: 会社情報（住所、TEL、FAX、社印等）および見積書のデフォルト設定に関する多数のカラムを追加。

### 2. 新規テーブルの追加
- **estimates**: 見積書のヘッダー情報（顧客、工期、金額、ステータス等）。
- **estimate_items**: 見積明細情報（項目名、数量、単価、金額、階層構造等）。

### 3. ロジック・ビューの解説追加
- **v_estimate_category_totals**: 見積カテゴリごとの集計ビュー。
- **トリガー**: 金額の自動計算や更新日時の自動更新ロジックの解説。
