# 現場選択のステータス制限実装計画

作業日報システム（WorkerApp）において、現場の選択ドロップダウンに表示される項目を、現在ステータスが「施工中」のもののみに制限します。

## ユーザーレビューが必要な事項

> [!IMPORTANT]
> ステータスが「完了」や「予定」の現場はドロップダウンに表示されなくなります。過去の日の日報を入力する場合でも、現在のステータスが「施工中」である必要があります。
> ただし、既にその日に入力済みの現場については、ステータスに関わらず上部のショートカットから選択・編集が可能です。

> [!NOTE]
> 「【会社】社内業務・雑務」という名称の現場については、日報入力の利便性を考慮し、ステータスに関わらず常に表示されるようにします。

## 検証プラン

### 手動確認
1. 管理者画面（AdminApp）で、いくつかの現場のステータスを「見積」「予定」「施工中」「完了」に設定する。
2. 作業員画面（WorkerApp）を開き、現場選択ドロップダウンを確認する。
3. 「施工中」の現場と「【会社】社内業務・雑務」のみが表示されていることを確認する。
4. すでに入力済みの（完了ステータスなどの）現場が、上部の「入力済みの現場」セクションに表示され、選択可能であることを確認する。
