# 現場選択のステータス制限 修正内容の確認

作業日報システムにおいて、現場の選択ドロップダウンの内容を制限する改修を行いました。

## 実施内容

### WorkerApp の修正

- `src/WorkerApp.jsx` において、現場選択のドロップダウンに表示される現場を以下の条件でフィルタリングするようにしました。
    1. ステータスが `施工中` である。
    2. または、現場名が `【会社】社内業務・雑務` である。

```javascript
// 修正後のロジック
{projects.filter(p => p.name === "【会社】社内業務・雑務").map(p => (<option key={p.id} value={p.id}>{p.name}</option>))}
{projects.filter(p => p.name !== "【会社】社内業務・雑務" && p.status === '施工中').map(p => (<option key={p.id} value={p.id}>{p.name || '無題の現場'}</option>))}
```

## 確認事項

- [x] 「見積」「予定」「完了」ステータスの現場が表示されないこと。
- [x] 「施工中」ステータスの現場が表示されること。
- [x] 「【会社】社内業務・雑務」が常に表示されること。
- [x] すでに入力済みの現場は、ステータスに関わらず上部のショートカットボタンから引き続き選択できること。
