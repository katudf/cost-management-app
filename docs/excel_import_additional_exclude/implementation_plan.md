# 実装計画：Excelインポート時の追加除外キーワード設定

## 背景と目的
現在、特定のキーワード（「安全費」「法定福利費」など）に基づいて間接費をExcelインポート時に除外していますが、ユーザーから提供された画像を確認したところ、その他にもシステムに取り込むべきでない項目（「諸経費」「値引き」など）が含まれていることが判明しました。これらについて、目標時間（人工）が発生する作業項目ではないため、インポート時に除外対象とするようロジックを更新します。

## 追加する除外キーワード
画像から確認できた以下の項目を、除外対象キーワードとして追加します。
- `諸経費`
- `値引き` (または `出精値引` などの表記揺れもカバーするため、`値引` をキーワードとする)

## App.jsx の修正箇所
`handleExcelImport` 内の既存のフィルタリング条件に、上記のキーワードを追加します。

```javascript
// 変更前
!name.includes('安全費') &&
!name.includes('法定福利費') &&
!name.includes('現場管理費') &&
!name.includes('共通仮設費') &&

// 変更後
!name.includes('安全費') &&
!name.includes('法定福利費') &&
!name.includes('現場管理費') &&
!name.includes('共通仮設費') &&
!name.includes('諸経費') &&
!name.includes('値引') &&
```

## ユーザーへの確認事項（User Review Required）
画像から「諸経費」「値引き」の2項目を新たに除外キーワードとして追加する方針としましたが、**その他にも普段お使いの見積書で「この行は作業実績（人工）として管理しないから除外してほしい」という定型的な項目名**はございますでしょうか？（例：「交通費」「処分費」「仮設トイレ」など）
現状判明しているものがあれば、このタイミングでまとめて追加いたします。
