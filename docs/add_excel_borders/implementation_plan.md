# 出力帳票（Excel）への罫線追加の実装計画

ユーザーからの要望である「出力帳票への罫線付与」を実現するための実装計画です。

## 背景と要件
現在 `src/AdminApp.jsx` にて、`xlsx` ライブラリを使用して「現場サマリー」と「作業項目別詳細」の2つのシートを持つExcelレポートを出力しています。
標準の `xlsx` ライブラリ（SheetJSのCommunity Edition）では、セルのスタイル（罫線、背景色など）を直接設定する機能が制限されています。
そのため、スタイル設定が可能なフォーク版である `xlsx-js-style` を導入し、出力時に罫線を引くように改修します。

## 提案する変更内容

### 1. 依存関係の追加
*   `xlsx-js-style` をインストールします。既存の `xlsx` に置き換わる形で使用可能です。

### 2. `src/AdminApp.jsx` の修正

#### [MODIFY] `src/AdminApp.jsx`
*   **インポートの変更:**
    ```javascript
    // 変更前
    // import * as xlsx from 'xlsx';
    
    // 変更後
    import * as xlsx from 'xlsx-js-style';
    ```
*   **`exportToExcel` 関数の改修:**
    *   既存の `json_to_sheet` で生成されたワークシート (ws1, ws2) に対して、各セルをループ処理し、スタイルオブジェクトを追加します。
    *   **適用するスタイル:**
        *   全てのセルに「細い実線（`thin`）」の上下左右の罫線（border）を設定します。
        *   （任意）1行目（ヘッダー行）に対しては、背景色（例：薄いグレー）や太字（bold）のスタイルを追加し、見栄えを良くします。

**スタイルオブジェクトの例:**
```javascript
const borderStyle = {
    top: { style: "thin" },
    bottom: { style: "thin" },
    left: { style: "thin" },
    right: { style: "thin" }
};
const headerStyle = {
    font: { bold: true },
    fill: { fgColor: { rgb: "EFEFEF" } }, // 薄いグレー
    border: borderStyle
};
```

## 検証計画

### 手動での検証
1.  **環境の準備:** `npm install xlsx-js-style` を実行後、ローカル開発サーバー (`npm run dev`) を起動します。
2.  **ブラウザでの操作:** AdminApp の画面（PC向け管理画面）を開き、任意の現場を選択します。
3.  **Excel出力:** 「Excel出力」ボタンをクリックし、ファイルが正常にダウンロードされることを確認します。
4.  **ファイル内容の確認:** ダウンロードされたExcelファイル（例: `現場名_工数管理レポート_YYYY-MM-DD.xlsx`）をExcel等の表計算ソフトで開きます。
5.  **罫線の確認:** 「現場サマリー」シートおよび「作業項目別詳細」シートのすべてのデータ領域のセルに、罫線が正しく引かれていることを目視で確認します。また、ヘッダーにスタイルが適用されていればそれも確認します。

## ユーザーへの確認事項
*   罫線の種類は一般的な「細い実線」でよろしいでしょうか。
*   ヘッダー行（1行目）を目立たせるために、薄いグレーの背景色と文字の太字化を追加してもよいでしょうか。これらを加えると、罫線と合わせてより帳票らしい見栄えになります。
