# 現場別エクセル帳票出力機能の追加 (Walkthrough)

## 実装した機能の概要
管理アプリ（AdminApp）の「管理シート」タブにおいて、現在選択されている現場の「サマリー（予測損益や全体の延べ人数）」と「作業項目別の詳細データ」をエクセル形式でダウンロードできる機能を追加しました。

## 変更点
- **AdminApp.jsx** に `exportToExcel` 関数を追加し、画面に表示されている計算結果を利用してエクセル出力用のデータを構築するようにしました。
- 延べ人数の計算は、ご要望の通り **「稼働時間 ÷ 8」** を自社作業員の1人工として算出し、協力業者の人数と合算して出力する仕様としています。
- 「管理シート」タブの**「項目別詳細予測」の右側に緑色の「Excel出力」ボタン**を配置しました。

## エクセル帳票のシート構成
出力されるエクセルファイルには以下の2つのシートが含まれます。

### 1. 現場サマリー
選択した現場の全体状況を出力します。
* 現場名
* 予測粗利
* 自社稼働時間合計(h)
* 自社延べ人数(人) ※時間÷8
* 協力業者延べ人数(人)
* 総・延べ人数(人) ※自社+協力業者
* 進捗率(%)

### 2. 作業項目別詳細
現場内の各作業タスクごとの状況を出力します。
* 現場名
* 作業項目名
* 進捗率(%)
* 目標工数(h)
* 消化工数(h)
* 自社延べ人数(人) ※時間÷8
* 予測粗利

## 動作確認の手順
1. 管理アプリにアクセスし、該当の現場を選択して **「管理シート」タブ** を開きます。
2. 画面中段の右側にある **「Excel出力」ボタン** をクリックします。
3. `(現場名)_工数管理レポート_YYYY-MM-DD.xlsx` というファイルがダウンロードされます。
4. ファイルを開き、2つのシートに期待通りのデータ（利益、延べ人数など）が出力されていることを確認してください。
