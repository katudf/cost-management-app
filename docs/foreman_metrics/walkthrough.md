# 職長ダッシュボード（指標表示）追加の確認

## 実装内容
作業員アプリ（WorkerApp.jsx）において、職長としてログインし、担当現場を選択した際に表示される「職長専用エリア」を拡張し、管理者画面と同等のダッシュボード指標を表示するようにしました。

### 追加された指標
緑色の「職長ダッシュボード」パネル内に以下の情報が表示されます。
1. **全体進捗率**: 現場の全作業における加重平均進捗率（%）
2. **消化工数 / 全体目標**: これまでに作業員全員が入力した合計実績時間 / 見積もりの全体目標時間
3. **現場全体の予測粗利**: （全体目標時間 - 予測完了時間）× システム共通の人工費用（デフォルト3500円） - 協力業者コスト

## 確認手順（手動テスト）

1. **ワーカーアプリにアクセス**
   - URL: `http://localhost:5173/worker` などを開きます。
2. **職長のアカウントでログイン**
   - 職長として登録されているユーザー名を選択します。
3. **担当現場の選択**
   - 自分が職長に割り当てられている現場を選択します。
4. **ダッシュボードの表示確認**
   - 画面上部の作業項目リストの直前に、緑色のヘッダーを持つ「職長ダッシュボード」パネルが表示されていることを確認します。
   - 「全体進捗率」「消化工数/全体目標」「予測粗利」の3つの数字が表示されているか確認してください。
5. **数値の整合性確認（任意）**
   - 管理者用アプリ（`AdminApp.jsx`）の「ホーム」または「管理シート」タブを開き、同じ現場の全体進捗率や予測粗利の金額と、ワーカーアプリに表示されている金額が一致しているか確認します。
6. **レイアウトの確認**
   - スマートフォンサイズの画面幅に縮小しても、パネル内の文字が隠れたりレイアウトが崩れたりせず、綺麗に表示されることを確認します。

## 課題・留意事項
- 目標時間が設定されていない（0時間の）作業項目が多い場合、進捗率や予測粗利が正しく計算・表示されない場合があります。これは管理者アプリと同様の仕様です。
