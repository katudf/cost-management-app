# AdminApp への見積書機能統合 - 修正内容の確認

`AdminApp.jsx` に対し、見積書機能（一覧および作成・編集画面）へのアクセスを可能にするための統合修正を行いました。

## 実施した変更内容

### 1. 新規コンポーネントのインポート
- `EstimateList` および `EstimateForm` を `AdminApp.jsx` の先頭でインポートしました。

### 2. 表示制御ステートの追加
- 見積一覧画面と詳細画面（新規・編集）の表示を切り替えるための `estimateEditId` ステートを導入しました。
    - `undefined`: 一覧表示
    - `null`: 新規作成
    - `number`: 編集

### 3. ナビゲーションメニューの更新
- ヘッダーのタブメニューに「見積」を追加しました。アイコンは `FileText` を使用しています。

### 4. レンダリングロジックの実装
- `main` コンテンツエリアにおいて、`activeTab` が `'estimate'` の場合の表示処理を追加しました。
    - `estimateEditId` が `undefined` の時は一覧画面（`EstimateList`）を表示。
    - それ以外の時は作成・編集画面（`EstimateForm`）を表示。
    - 保存完了時や「戻る」ボタン押下時に一覧に戻るコールバックを設定しました。

## 検証結果

- [x] コードの構文エラーがないことを確認。
- [x] インポートパスがディレクトリ構造と一致していることを確認（`src/EstimateList.jsx`, `src/EstimateForm.jsx`）。
- [x] ステート名が既存のコードと衝突していないことを確認。
