# AdminApp への見積書機能統合計画

`AdminApp.jsx` に新しく作成された見積書機能（一覧表示および編集フォーム）を統合します。

## ユーザーレビューが必要な事項

> [!IMPORTANT]
> **コンポーネントの存在確認済み**
> `src/EstimateList.jsx` および `src/EstimateForm.jsx` が既に存在することを確認しました。これらのインポートによりアプリに統合されます。

## 提案される変更内容

### [AdminApp.jsx](file:///c:/Users/katuy/Desktop/cost-management-app/src/AdminApp.jsx) の修正

#### 【変更1】インポートの追加
- `EstimateList` および `EstimateForm` をインポートします。

#### 【変更2】ステートの追加
- 見積一覧と編集フォームの表示状態を制御するための `estimateEditId` ステートを追加します。
    - `undefined`: 一覧表示
    - `null`: 新規作成
    - `number`: 編集

#### 【変更3】ナビゲーションタブの追加
- ヘッダーのナビゲーションメニューに「見積」タブを追加します。

#### 【変更4】コンテンツ表示ロジックの追加
- `activeTab` が `'estimate'` の場合に、`estimateEditId` の状態に応じて一覧またはフォームをレンダリングするロジックを `main` タグ内に追加します。
