# 作業員配置表 — 実装記録

## フェーズ1: 配置表タブ・工期フィールド

- `AssignmentChartTab.jsx` — 案件バーチャート + 作業員日別配置グリッドを新規作成
- `MasterTab.jsx` — 工期開始日・終了日フィールドを工事設定タブに追加
- `AdminApp.jsx` — 「配置表」タブをナビゲーションに追加
- 土曜=青、日曜=赤のヘッダー色分け
- `Projects.startDate` / `endDate` に工期を保存

## フェーズ2: セル編集（配置追加・削除）

- セルクリックでポップアップ表示 → 案件を選択して配置追加
- ゴミ箱ボタンで配置削除
- 1日に複数案件の配置に対応
- **楽観的更新（Optimistic Update）** — DB通信を待たずUI即時反映、ラグ解消

## フェーズ3: その他スケジュール＆カラー設定

- 現場以外の予定（有給、休み、健診、会社、講習会）を配置可能に
- `Assignments` テーブルの `projectId=null` / `title=種別名` で既存構造を流用
- `MasterTab.jsx` にカラーパレット（9色プリセット）追加
- `Projects.bar_color` に保存、バーチャートとセルに即時反映

## フェーズ4: ドラッグ選択＆週コピー

- **ドラッグ選択**: セルをクリック＆ドラッグで同一作業員の複数日を一括選択
  - ポップアップに「03/31 〜 04/02 (3日)」のように範囲表示
  - `addAssignmentBatch` で一括INSERT
- **週コピー**: ヘッダーの「週コピー」ボタンからモーダル表示
  - コピー元/先の週をプルダウン選択
  - 対象作業員を全員または個別に選択可能
  - 重複配置はスキップ

## フェーズ5: Excel出力

- ヘッダーに「Excel出力」ボタン（`Download`アイコン）を追加
- `src/utils/assignmentChartExport.js` — Excel生成ロジック
  - バーチャート行（案件色付き）、作業員配置行（配置色付き）
  - 土日の背景色、ヘッダーのスタイリング
- Blob + aタグ方式でダウンロード（ブラウザ互換性確保）
- ファイル名: `assignment_chart_YYYY-MM-DD.xlsx`

## フェーズ6: 会社休日設定 (マルチセレクト機能付き)

- システム設定タブ（`SystemSettingsTab.jsx`）内に「会社休日設定」セクションを追加
- `CompanyHolidays` テーブルと連携し、休日の新規追加、一覧表示、削除ができるように実装
- 単一の `<input type="date">` から **マルチセレクトカレンダーUI** に改修
    - 1ヶ月のカレンダーグリッドから、休日としたい日付を複数クリックして選択可能
    - 「〇件をまとめて追加」ボタンで、選択したすべての日付を一括で登録
    - すでに登録済みの休日はカレンダー上で赤く表示され、重複登録を防止
    - 追加時のデフォルトの文言を「会社休業日」に設定し、一括で任意の説明も登録可能
- **配置表タブ（`AssignmentChartTab.jsx`）での休日カラー反映機能を実装**
    - `CompanyHolidays`から取得した休日を対象とし、対象日のヘッダー背景色と文字色を日曜日と同じ「赤色」で表示するよう色分け処理（`getDayHeaderStyle`等）を修正
    - 各作業員のセルについても、休日扱いとして「灰色のハイライト背景（#F9FAFB）」が適用されるよう対応

## セル幅固定

- 配置セルの `td` に `overflow: hidden` / `maxWidth: 0` / `width: 48px` を設定
- 内容の長さに関係なくセル幅を固定

## スマホ用配置予定表ビュー

- `src/ScheduleViewApp.jsx` — 新規コンポーネント
- アクセス: `?mode=schedule` をURLに追加
- `App.jsx` にルーティング追加
- **表示内容**: 作業員名＋配置現場名のグリッドのみ（バーチャート・管理タブ非表示）
- 今日の列を青枠でハイライト、2週間表示、読み取り専用
