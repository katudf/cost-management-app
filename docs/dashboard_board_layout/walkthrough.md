# ホームタブダッシュボードのレイアウト変更

ご要望通り、ホームタブの工事一覧を「見積」「予定」「施工中」「完了」のステータスごとに4列で表示するカンバン形式のレイアウトに刷新しました。

## 実施した変更内容

### 1. 4列カンバン表示の導入
- 旧来のカードの一元的なグリッド表示を廃止し、ステータス別に工事を分類して横並びに表示するUIに変更しました（`AdminApp.jsx`）。
- 画面幅が狭い場合（スマートフォン等）は縦に積み重なり、PC画面では4列が横並びになるレスポンシブデザインに対応しています。

### 2. 並べ順の「新しい順」への固定
- 工事の並び順が指定通り「作成または更新の日付が新しい順」になるよう、内部ロジック（`useDashboardStats.js`）を固定しました。
- これに伴い、上部にあった「並び順」のドロップダウンメニュー、および「表示対象」の絞り込みチェックボックスを削除し、UIをすっきりと整理しました。全ステータスの全件がそれぞれの列に分類されて表示されます。
- 工事名によるテキスト検索機能は引き続きご利用いただけます。

### 3. 最大10件表示とページネーション機能
- 各ステータスの列には、最大10件までの工事カードのみを表示するように制限を設けました。
- 11件以上存在するステータスの列においては、自動的にページ送りのナビゲーション（「< 前」「次 >」ボタンとページ番号）が表示されるロジックを追加しています。

## 検証結果

- [x] すべてのプロジェクトが該当するステータスの列に分類されて表示されること
- [x] ヘッダーコントロールが撤去され、UIがクリーンになっていること
- [x] プロジェクト数が多い場合に10件で分割され、ページ送りが正しく機能すること
