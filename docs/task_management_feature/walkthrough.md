# 実績・確認内容: 工事設定画面のタスク追加・削除機能

## 達成したこと
「工事設定」タブにおいて、Excelインポートを使用しなくても、画面上から手動で作業項目（タスク）を新規追加したり、不要な項目を削除できる機能を実装しました。これにより、現場運用中の細かなタスクの調整が可能になりました。

## 変更されたファイルと内容
- **`src/App.jsx`**
  - 新規タスク追加用の `addMasterItem` 関数を実装しました。この関数は、関連する現場ID（Project ID）を取得し、Supabaseの `ProjectTasks` テーブルへ新しい行をインサートします。
  - タスク削除用の `removeMasterItem` 関数を実装しました。この関数は、実行前に確認ダイアログを表示し、承諾された場合にのみ、Supabaseの該当データを削除し、ローカルの `masterData` と `records` からも関連データをクリーンアップします。
  - 工事設定タブのUIを更新し、各タスクの行の右端に「ゴミ箱アイコン（削除ボタン）」を追加、またタスク一覧の最下部に「＋ 新しい作業項目を追加」ボタンを設置しました。

## ローカル環境でのテスト結果
- **UI表示**: 「工事設定」タブを開いた際、タスクの各行に削除ボタンが表示され、リスト下部に追加ボタンが表示されることを確認しました。
