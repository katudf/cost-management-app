# 協力会社・材料・作業手当セルの縦分割レイアウト実装計画

協力会社、使用材料、および作業手当のセクションを、1行にまとめつつ「項目名」と「数量/サイン」を縦に分割して表示するように修正します。

## 概要
これまで複数行に分かれていた項目を1行に集約し、各日のセル内をさらに縦に区切るレイアウトに変更します。

## ユーザー承認済み事項
- 使用材料セクションも協力会社と同様の縦分割レイアウトに変更する。
- **追加**: 作業手当セクション（時間(H)・承認サイン）も同様の縦分割レイアウトに変更する。
- 全体のカラム構造を倍増させ、他の行には `colspan="2"` を適用して見た目を維持する。

## 修正内容

### 1. カラム構造の再定義
表全体の列構成を以下のように変更（合計17物理列）。
- **列1**: グループラベル [1列]
- **列2〜3**: 項目詳細ラベル（分割用） [2列]
- **列4〜17**: 日付データ欄（7日分 × 2列） [14列]

### 2. コンポーネント別の変更
#### `generateWorkerReportPDF` 内部ロジック
- **ヘッダー**: 各日付 `<th>` に `colspan="2"` を付与。
- **協力会社・使用材料行**: 1行に統合し、左側を名称、右側を数量として縦分割。
- **作業手当行 [今回の追加]**: 
    - 「時間(H)」と「承認サイン」を1行に統合。
    - 各日のセル内を左右に分け、左に時間、右にサイン欄を配置。
- **現場・日計・備考**: `colspan="2"` を適用。

### 3. スタイル定義の更新
- **`.vertical-label`**: 項目名を縦書きにするスタイルを追加。
- **破線の追加**: 分割用の破線スタイルを維持。

## 検証項目
- [ ] 作業手当が1行になり、左右に分割されているか。
- [ ] 協力会社・使用材料のレイアウトが正しく維持されているか。
