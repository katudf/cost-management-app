# 見積金額表示機能 実装計画

## 背景・目的
ユーザーからの要望により、**「管理シート（詳細画面）」と「工事設定」のタブにおいて、各作業項目の見積金額を表示**する機能を追加します。
現状、見積から算出した「目標時間（工数）」はありますが、「元の見積金額」自体は保持・表示されていませんでした。

## 課題
現在、Excelインポート時（`handleExcelImport`）に、エクセルの金額から目標工数（時間）を `amount / HOURLY_WAGE` (4375円ベース) で計算していますが、元の金額自体は状態（`masterData`）やデータベース（`ProjectTasks`）に保存されていません。

## 提案する実装内容 (Proposed Changes)

### 1. データベース (Supabase) の変更
*   **[MODIFY] `ProjectTasks` テーブル**
    *   各作業項目に見積金額を保存するためのカラム `estimated_amount` (numeric型) を追加します。

### 2. AdminApp.jsx の改修 (データの保存とフロー)
*   **[MODIFY] `src/AdminApp.jsx`**
    *   **Excelインポート処理 (`handleExcelImport`)**: エクセルから読み取った金額 (`amount`) を、目標時間だけでなく、新たに `estimated_amount` としても保持し、DB（`ProjectTasks`）のインポート処理時に追加します。
    *   **DB読込処理 (`fetchAllData`)**: `ProjectTasks` から `estimated_amount` も取得し、ローカルステートの `masterData` オブジェクトに含めます。
    *   **手動追加・編集処理**: 「工事設定」タブで作業項目を直接追加・修正する際にも、「見積金額」を入力・変更・DB保存できるようにします。

### 3. UIの変更 (AdminApp.jsx)
*   **[MODIFY] 工事設定タブ (`activeTab === 'master'`)**:
    *   各作業項目の行に「見積金額」の入力ボックス列を新設します。
*   **[MODIFY] 管理シートタブ (`activeTab === 'summary'`)**:
    *   「項目別詳細予測」のテーブルに「見積金額」の列（表示専用）を新設します。必要に応じて全体の合計金額サマリーを出すことも検討します。
*   **[MODIFY] エクセル出力機能 (`exportToExcel`)**:
    *   出力されるエクセルにも「見積金額」列を追加します。

## 検証計画 (Verification Plan)
### 手動検証 (Manual Verification)
1. **DBカラム追加の確認**: SQLエディタで `ProjectTasks` テーブルに `estimated_amount` が追加されていることを確認する。
2. **Excelインポート動作確認**: AdminAppの「工事設定」タブからExcelファイルをインポートし、「見積金額」が正しく画面（管理シート・工事設定）に反映されるか確認する。
3. **手動入力・編集の確認**: 「工事設定」タブで「見積金額」を手入力・修正し、リロードしてもDB経由で正しく反映されるか確認する。
