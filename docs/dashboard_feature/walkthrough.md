# 実績・確認内容: ダッシュボード（工事一覧）画面の追加

## 達成したこと
全現場を一覧で俯瞰できる「ダッシュボード（ホーム）」機能の実装を完了しました。

## 変更されたファイルと内容
- **`src/App.jsx`**
  - アプリの初期表示タブ（`activeTab`）を `'summary'` から `'dashboard'` に変更し、起動時にダッシュボードが表示されるようにしました。
  - 画面上部のタブメニューの先頭に「ホーム」を追加しました。
  - 新たに `allProjectsSummary` という集計用の関数を作成し、各プロジェクトに紐づくすべてのタスクと実績から「合計目標工数」「合計実績工数」「全体進捗率」「予測粗利」を算出するロジックを組み込みました。
  - メインエリアに、取得したプロジェクトの情報をカード型で表示するレイアウトを実装しました。
  - 各プロジェクトカードをクリックすることで、選択状態をその現場に切り替え、自動的に「管理シート」タブへ遷移する導線を実装しました。

## ローカル環境でのテスト結果
- **UI表示**: アプリ起動時に「登録済み工事一覧」画面（ホーム）が正しく表示されることを確認しました。
