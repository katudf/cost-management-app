# 見積書 PDF ダウンロード機能統合 - 修正内容の確認

`EstimateList.jsx` に対し、見積書を PDF としてダウンロードし保存できる機能を統合しました。

## 実施した変更内容

### 1. 必要なモジュールとアイコンのインポート
- `EstimatePDF.jsx` から `downloadEstimatePDF` 関数をインポートしました。
- `supabase` クライアントを正しいパス（`./lib/supabase`）からインポートしました。
- `lucide-react` から `Download` アイコンを追加しました。

### 2. PDF ダウンロードハンドラの追加
- `handleDownloadPDF` 関数を `EstimateList` コンポーネント内に追加しました。
- この関数は、選択された見積の詳細データ（明細含む）と、PDF に記載する自社情報（`system_settings`）を Supabase から取得し、PDF 生成・ダウンロード処理を実行します。

### 3. 一覧画面の UI 更新
- `EstimateRow` コンポーネントに `onDownload` プロップを追加し、PDF 出力ボタンを表示するようにしました。
- 操作列の左端に緑色のダウンロードアイコンが表示されます。

## 検証結果

- [x] インポートパスがプロジェクト構造と一致していることを確認（`src/lib/supabase.js`）。
- [x] 編集ボタンの左側に PDF 出力ボタンが正しく配置されていることを確認。
- [x] ステートとプロップの受け渡しが正しく実装されていることを確認。

---

> [!TIP]
> 見積一覧の「操作」列にある緑色のアイコンをクリックすると、その見積書の PDF が生成されダウンロードされます。自社情報の社名や連絡先も自動的に反映されます。
