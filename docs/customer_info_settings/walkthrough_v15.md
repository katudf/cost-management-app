# 工期未設定プロジェクトのクイック設定機能 - 修正内容の確認

配置表中において、工期がまだ決まっていない工事についても表示し、カレンダー上をクリックするだけで仮の工期を設定できる機能を追加しました。

## 実施した変更内容

### 1. 未設定プロジェクトの可視化
- **[AssignmentChartTab.jsx](file:///c:/Users/katuy/Desktop/cost-management-app/src/components/tabs/AssignmentChartTab.jsx)**
    - これまでは工期（開始日・終了日）が両方設定されている工事のみを表示していましたが、ステータスが「予定」または「施工中」であれば、工期未設定の工事もリストに表示するようにクエリを修正しました。

### 2. セルクリックによる工期設定
- **[AssignmentChartTab.jsx](file:///c:/Users/katuy/Desktop/cost-management-app/src/components/tabs/AssignmentChartTab.jsx)**
    - 工期未設定の行の空白セルにホバーすると、薄く青色が付き、クリック可能な状態（`cursor-pointer`）になるようにUIを改善しました。
    - **動作**: 空白セルをクリックすると、その日を「開始日」および「終了日」として即座にDBへ登録します。
    - 設定された後は、ドラッグ操作によって期間を左右に伸ばしたり、位置を動かしたりすることが可能です。

## 確認事項

- [x] 配置表の左側のリストに、まだ工期を設定していない工事名が表示されている。
- [x] その工事の行の、カレンダー上のセルをクリックするとバーが表示され、「工期を設定しました」とトースト通知が出る。
- [x] 設定後、バーの端をドラッグして期間を変更できる。

> [!TIP]
> 案件だけ先に登録しておき、配置表を見ながら直感的にいつから始めるかを決めるフローがスムーズになりました。
