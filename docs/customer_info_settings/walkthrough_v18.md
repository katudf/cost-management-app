# 配置表の表示移動ステップ変更 - 修正内容の確認

配置表のヘッダーにある前後移動ボタン（左右の矢印）の移動距離を、従来の8週間単位から1週間単位に変更しました。

## 実施した変更内容

### 1. 移動ステップの調整
- **[AssignmentChartTab.jsx](file:///c:/Users/katuy/Desktop/cost-management-app/src/components/tabs/AssignmentChartTab.jsx)**
    - 前後移動ボタンをクリックした際の日数調整を 56日（8週）から 7日（1週）に変更しました。
    - ボタンのホバー時に表示されるヒント（ツールチップ）を「前の週」「次の週」に更新しました。

## 確認事項

- [x] 配置表ヘッダーの左矢印ボタンを押すと、表示期間がちょうど1週間前（月〜日）に移動する。
- [x] 右矢印ボタンを押すと、表示期間がちょうど1週間後に移動する。
- [x] 「今日」ボタンを押すと、現在の日付を含む週の月曜日に戻る（従来通りの動作）。

> [!TIP]
> 1週間単位での移動になったことで、直近の微調整や週単位でのスケジュール確認がより正確に行えるようになりました。
