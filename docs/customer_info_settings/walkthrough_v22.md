# 現場名からの工事基本設定ジャンプ機能 - 修正内容の確認

配置表の左側にある現場名をクリックすることで、その工事の詳細設定（受注・仕様項目設定）画面へ即座に移動できる機能を追加しました。

## 実施した変更内容

### 1. クリックイベントの追加
- **[AssignmentChartTab.jsx](file:///c:/Users/katuy/Desktop/cost-management-app/src/components/tabs/AssignmentChartTab.jsx)**
    - 現場名のセル（2段表示になっている部分）にクリックイベントを追加しました。
    - クリックすると、該当プロジェクトが選択された状態で「工事基本設定（マスター）」タブへ自動的に切り替わります。

### 2. UIの改善
- 現場名にマウスを合わせた際、クリック可能であることがわかるようにカーソルを指マーク（pointer）に変更し、背景色が変わるようにしました。

## 確認事項

- [x] 配置表の現場名（または顧客名部分）をクリックすると、画面が切り替わり、その現場の基本設定画面が表示される。
- [x] マウスを合わせた時に背景が薄い青色になり、クリックできることが直感的にわかる。

> [!TIP]
> 配置表でスケジュールを確認しながら、気になった現場の予算や仕様をすぐに確認・編集できるようになり、画面間の移動がスムーズになりました。
