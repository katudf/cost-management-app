# 休日カレンダーのレイアウト調整 - 修正内容の確認

会社休日カレンダーのタイル表示のレイアウトを調整しました。

## 実施した変更内容

### 3列4行のグリッドレイアウトへの変更
- **[HolidayCalendar.jsx](file:///c:/Users/katuy/Desktop/cost-management-app/src/components/HolidayCalendar.jsx)**
    - カレンダーの月タイルの表示を、画面サイズに関わらず「3列（3 columns）」で固定するように修正しました。
    - これにより、全12ヶ月が「3列 × 4行」の整然としたタイル形式で並ぶようになり、年度全体の構成が見やすくなりました。

## 確認事項

- [x] 設定タブの「休日設定」において、カレンダーが横に3つ並んでいる。
- [x] 縦に4段（合計12ヶ月分）並んで表示されている。

> [!NOTE]
> 3列固定にすることで、1つ1つの月のカレンダーに適度な幅が確保され、統計情報（休日・出勤日数）も読みやすくなりました。
