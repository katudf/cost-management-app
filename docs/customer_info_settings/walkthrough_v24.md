# 休工期間の視覚化機能 - 修正内容の確認

長期現場等で工事期間中に現場に入らない期間（休工期間）を、ガントチャート上で斜線パターンにより視覚的に表示する機能を実装しました。

## 実施した変更内容

### 1. 工事基本設定への休工期間管理UI追加
- **[MasterTab.jsx](file:///c:/Users/katuy/Desktop/cost-management-app/src/components/tabs/MasterTab.jsx)**
    - 工期設定の下に「休工期間」セクションを新設しました。
    - 開始日・終了日・理由（任意）を入力して「追加」ボタンで登録できます。
    - 登録済みの休工期間はオレンジ色のカードで一覧表示され、✕ボタンで個別に削除可能です。
    - 1つの現場に複数の休工期間を設定できます。

### 2. 配置表（ガントチャート）での斜線パターン表示
- **[AssignmentChartTab.jsx](file:///c:/Users/katuy/Desktop/cost-management-app/src/components/tabs/AssignmentChartTab.jsx)**
    - 配置表のデータ取得時に `ProjectSuspensions` テーブルからも休工期間を取得するように拡張しました。
    - ガントチャート上で休工期間に該当するセルに、CSSの `repeating-linear-gradient` を使用した**45度の斜線パターン**を表示します。
    - 斜線パターンはプロジェクトのバー色をベースに、薄い色合いで描画されるため、工期の範囲内であることは維持しつつ「休工中」であることが一目でわかります。
    - セルにマウスを合わせると「休工期間」というツールチップが表示されます。

### 3. データベース
- **`ProjectSuspensions` テーブル (新規作成)**
    - `project_id`: 対象プロジェクト（Projects テーブルへの外部キー、CASCADE削除）
    - `start_date` / `end_date`: 休工の開始日・終了日
    - `reason`: 理由（任意）

## 確認事項

- [x] 工事基本設定（マスタータブ）で休工期間の追加・削除ができる。
- [x] 配置表のガントチャート上で、休工期間に該当する日のセルが斜線パターンで表示される。
- [x] 通常の稼働日はベタ塗り、休工期間は斜線と、視覚的に区別できる。
- [x] セルにマウスを合わせると「休工期間」とツールチップが出る。

> [!TIP]
> 休工期間は工期の範囲内に限り表示されます。工期外の日付に設定しても配置表上では反映されません（意味のないデータとして無視されます）。
