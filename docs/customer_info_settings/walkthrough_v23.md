# 配置表の現場名への通し番号追加 - 修正内容の確認

配置表の左側に表示される現場名の先頭に、情報の整理や場所の特定を容易にするための通し番号（1. 2. 3...）を追加しました。

## 実施した変更内容

### 1. 通し番号の表示
- **[AssignmentChartTab.jsx](file:///c:/Users/katuy/Desktop/cost-management-app/src/components/tabs/AssignmentChartTab.jsx)**
    - 配置表に表示されている案件を上から順番にカウントし、番号を表示するようにしました。
    - 番号はフォントサイズを小さめ（10px）にし、薄いグレー（slate-400）で表示することで、現場名そのものの視認性を邪魔しないように配慮しました。
    - 数字の幅を固定（w-4）しているため、1桁と2桁が混在しても現場名の開始位置が揃うようになっています。

## 確認事項

- [x] 配置表の各現場名の左側に、通し番号（1 2 3...）が表示されている。
- [x] ドラッグ＆ドロップで並び順を変更した際、番号は常に上から「1 2 3...」と振り直される。

> [!NOTE]
> 番号が付与されたことで、「上から5番目の現場」といった口頭でのコミュニケーションや、全体件数の把握がより容易になりました。
