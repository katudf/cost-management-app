# 巨大コンポーネントのリファクタリング（責務の分解）実装計画書

## 目的
1ファイルに多大なロジックが集中している `AssignmentChartTab.jsx` (~1,800行) と `SystemSettingsTab.jsx` (~970行) を、役割ごとにカスタムフックおよびサブコンポーネントへ分割します。
これにより、コードの可読性を大幅に向上させ、機能追加時のバグ混入リスクを軽減し、メンテナンス性を向上させます。

## 影響分析

1. **設計ポリシー（Design Policies）との整合性**:
   * `docs/architecture.md` の「ビジネスロジックの分離」および「コンポーネント階層の維持」に整合します。
   * React状態管理とUI描画コードを分離します。

2. **重要経路（Critical Paths）への副作用**:
   * **配置表（`AssignmentChartTab`）**: 状態変更（ドラッグによるアサイン期間変更、コピペ、削除）が Supabase データベースと同期しており、さらにリアルタイム通信（リアルタイムチャネル）で購読されています。リファクタリングの過程でこれらの状態更新処理が崩れないよう、ロジックを慎重にカスタムフックへカプセル化します。
   * **設定（`SystemSettingsTab`）**: 各サブ設定は完全に独立した機能であるため、他タブ（ホーム、工事設定など）への影響は軽微ですが、資格情報の追加・削除が作業員マスタ全体へ連動しているため、再フェッチ (`fetchAllData`) が正しく機能し続けることを担保します。

3. **デグレードの防止**:
   * アサイン操作（ガントチャートのドラッグ移動・リサイズ、セルのコピペ、キーボードによる削除操作）が、リファクタリング前と全く同一の操作感・挙動で動くことを確認します。

## 提案される変更点

### 1. 配置表 (`AssignmentChartTab`) のリファクタリング

#### [NEW] `src/hooks/useAssignmentState.js`
配置表の全ロジック（データ取得、リアルタイム通信、アサイン追加・一括追加・削除、コピー・カット・ペースト等のクリップボード操作、ガントチャートドラッグ操作）をカプセル化したカスタムフック。

#### [NEW] `src/components/assignment/` ディレクトリの作成
UI描画のための各種ポップアップコンポーネントを切り出します。
* [NEW] `src/components/assignment/EditColorPopup.jsx` （プロジェクトカラー編集）
* [NEW] `src/components/assignment/EditHolidayPopup.jsx` （会社休日セル編集）
* [NEW] `src/components/assignment/AssignmentPopup.jsx` （配置セルの編集ポップアップ：コピー、ペースト、カット、削除などのメニュー）

#### [MODIFY] `src/components/tabs/AssignmentChartTab.jsx`
* UI描画（グリッド・テーブル表示、ガントチャート部分）に専念させ、ロジックは `useAssignmentState` から受け取る形にリファクタリングします。
* ポップアップ表示を新規コンポーネントに委譲します。
* これにより、約1,800行から **500行程度** へと大幅に軽量化します。

---

### 2. システム設定 (`SystemSettingsTab`) のリファクタリング

#### [NEW] `src/components/tabs/settings/CertificationManager.jsx`
* `SystemSettingsTab.jsx` 内に直接記述されている「資格情報マスター」（資格の追加・編集フォーム、作業員別・資格別のテーブル表示）を別コンポーネントへ分離。

#### [NEW] `src/components/tabs/settings/CompanyInfoSettings.jsx`
* 「自社情報」（自社名、住所、電話、社印・代表印画像のアップロード処理と保存）の設定部分を別コンポーネントへ分離。

#### [MODIFY] `src/components/tabs/SystemSettingsTab.jsx`
* サブタブによる表示切替のみを行い、設定本体のUIはそれぞれのコンポーネント（`HolidayCalendar`, `CustomerSettings`, `StaffSettings`, `CertificationManager`, `CompanyInfoSettings`）をインポートして呼び出すルーティングのみを担当するように整理します。
* これにより、約970行から **150行程度** へと軽量化し、極めて可読性の高い状態にします。

## 検証計画

### 動作検証
1. **アサイン管理の動作確認**:
   * 配置セル（一般作業員・職長）をドラッグして複数日にアサインできること。
   * アサイン済みのセルを選択し、`Ctrl + C`（コピー）、`Ctrl + V`（ペースト）、`Delete`（削除）がキーボードおよびメニューポップアップから同一に機能すること。
   * ガントチャート（案件バー）の左右端をドラッグして工期を変更でき、Supabase が更新されること。
   * 他のブラウザやタブで変更したアサイン内容が、リアルタイム通信（リアルタイム更新）で即座に同期されること。
   * Excel エクスポートが正常に動作すること。
2. **システム設定の動作確認**:
   * 「資格情報マスター」での資格追加、資格の編集、削除が正常に行え、作業員別・資格別の表示切り替えが正しく動くこと。
   * 「自社情報」の入力・保存、および代表印・社印画像のアップロードが正常に終了し、見積書PDFに出力されること。
