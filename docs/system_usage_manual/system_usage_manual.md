# 原価管理システム（スマホ・PC対応） 操作マニュアル

本システムは、現場作業員向けの「作業員アプリ（スマホ用）」と、管理者・事務・営業向けの「管理者アプリ（PC推奨）」の2つで構成されています。
それぞれの役割に応じた入力と確認を行うことで、リアルタイムな現場の進捗管理と原価把握が可能になります。

---

## 1. 役割と担当機能の一覧

| 役割 | 担当 | アプリ | 機能・タブ |
| :--- | :--- | :--- | :--- |
| **営業・営業推進** | 新規案件の登録・見積取込 | 管理者 | **「工事設定」タブ** |
| **事務員** | 協力業者の実績入力、過去データの修正 | 管理者 | **「実績入力」タブ** |
| **職長 (現場責任者)** | 担当現場の進捗率入力 | 作業員 | **(現場・進捗画面)** / または 管理者**「ダッシュボード」タブ** |
| **作業員 (全般)** | 日々の作業時間・内容の入力 | 作業員 | **本日の実績・日報入力画面** |
| **管理者・経営層** | 現場の損益確認・日報出力 | 管理者 | **「ダッシュボード」タブ, 「作業員」タブ** |

---

## 2. 管理者アプリ（PC推奨）の操作方法

アクセスURL: （管理者用URLを開いてください。例: `https://.../`）

管理者アプリには以下の4つの主要タブがあります。

### ① 工事設定（マスター・仕様）タブ
**主な担当者:** 営業・営業推進
新規現場が受注された際や、見積もりが作成された際に行う初期設定画面です。

1. **現場の追加**: 新しい現場名を登録し、「現場ステータス」を「予定」や「施工中」に設定します。
2. **職長の割り当て**: その現場の責任者（職長）をプルダウンから選択します（※これを設定すると、職長がスマホから進捗率を入力できるようになります）。
3. **見積データ（Excel）の取り込み**:
   - 右上の「Excelからインポート」ボタンを押し、見積書のExcelファイルを読み込ませます。
   - 取り込まれた「見積金額」から、自動的に**「目標時間（時間）」**が算出されます（見積金額 ÷ 4,375円 で計算されます）。
   - 必要に応じて手入力で作業項目や金額を追加・修正することも可能です。

### ② 実績入力タブ
**主な担当者:** 事務員
作業員がスマホから入力した日報データがここに集約されます。また協力業者の入力もここで行います。

1. **自社作業員の日報**:
   - スマホから入力されたデータは自動的にここに反映されます。
   - 事務員は必要に応じて「日付」「作業項目」「入力時間」「備考」の修正・削除を行えます。
   - 追加漏れがある場合は「日報1件追加」から手動で登録することもできます。
2. **協力業者の実績入力**:
   - 協力業者の入った日付、会社名、人数を手入力で追加し管理します。

### ③ ダッシュボード（サマリー）タブ
**主な担当者:** 職長・管理者
現場全体の「予測粗利」や「消化済工数」、各作業項目の進捗を確認するメイン画面です。

1. **進捗率の更新 (職長・管理者)**:
   - 各作業項目の「進捗」バー（スライダー）を動かして、現在の進捗率（0〜100%）を入力します。
   - 進捗率と今までに入力された実績工数をもとに、**工数差異（プラスマイナス）」や「項目別予測粗利」がリアルタイムに自動計算**されます。
2. **現場全体の確認**:
   - 画面上部に、このままのペースで進んだ場合の「現場全体の予測粗利」が表示されます。赤色で点滅している場合は赤字予測となっているため、至急の対策が必要です。
3. **Excel出力**:
   - 右上の「Excel出力」ボタンから、現在表示中のサマリーデータをExcel形式でダウンロードして報告書等に活用できます。

### ④ 作業員管理タブ
**主な担当者:** 事務員・管理者
自社作業員のマスターデータ管理と、各作業員の総稼動時間の確認・日報出力を行います。

1. **作業員マスター設定**:
   - 「追加する」から新入社員などを登録できます（名前・フリガナ・生年月日など）。
   - ここで表示されている矢印ボタン（↑↓）で並び替えた順番が、**スマホの「作業員アプリ」のログイン選択画面の順番に反映**されます。
2. **稼働集計と日報出力**:
   - 各作業員がどの現場で何時間稼働したかの総計が確認できます。
   - 対象期間を選んで「出力」ボタンを押すと、その人の**勤怠・作業日報（通常・残業区別あり）がExcel形式でダウンロード**できます。

## 3. 作業員アプリ（スマホ推奨）の操作方法

アクセスURL: （作業員用URLを開いてください。例: `https://.../worker`）

現場の作業員は、毎日作業終了後にスマホから日報データを入力します。

### ① ログイン画面
1. 自分の名前をリストから探してタップ（または検索して選択）します。
2. （職長の場合）「進捗入力する現場を選択」画面が表示されるので、現場をタップして先に進みます。一般作業員の場合は直接「日報入力画面」へ進みます。

### ② 現場の進捗入力 ※職長のみ
- 自分が担当に設定されている現場の作業項目一覧が表示されます。
- 作業が終わった項目や進んだ項目の「+」「-」ボタン（またはスライダー）を使って、現在の進捗率（％）を更新し、「進捗を更新する」ボタンを押します。

### ③ 本日の日報入力（全作業員共通）
1. **現場と作業の選択**:
   - 「現場追加」ボタンを押し、今日作業した現場と、メインで行った作業項目を選びます。
2. **作業時間の入力**:
   - 今日その現場で働いた「通常時間（例：8h）」と「残業時間（例：2h）」を入力します。
3. **備考の入力**:
   - 特記事項や連絡事項があれば「備考」に記入します。
   - ※複数現場に行った場合は、再度「現場追加」を押して入力欄を増やし、分けて入力してください。
4. **送信**:
   - 画面一番下の**「本日の実績を〇〇時間 送信する」**ボタンを押すと入力完了です。入力されたデータは、管理者の「実績入力」タブへ反映されます。

---

## 4. 日々の運用フロー（まとめ）

1. **【営業】** 新規現場が決まったらPCで「①工事設定」から現場と見積書を登録し、職長を設定。
2. **【作業員】** 毎日作業終わりに、スマホから「③日報入力」を行う（各種時間を入れる）。
3. **【職長】** 毎日（または定期的に）、スマホ（またはPCのダッシュボード）から担当現場の「進捗率（％）」を更新する。
4. **【事務員】** 協力業者が入った日は、PCの「②実績入力」に協力業者情報を入力する。
5. **【管理者・職長】** PCの「③ダッシュボード」を見て、現在のペースで赤字にならないか（予測粗利）をチェックし、現場の対策を立てる。
6. **【事務員】** 月末や締め日に、PCの「④作業員管理」から出勤日報をExcel出力して勤怠や給与計算に回す。
