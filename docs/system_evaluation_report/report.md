# システム評価用レポート（詳細工数管理システム）

## 1. システムの概要
本システムは、建設業などの現場作業における工数（労働時間）とそれに伴うコスト（労務費）をクラウド上でリアルタイムに管理・可視化するためのWebアプリケーションです。
管理者向けの「管理アプリ（Admin App）」と、現場で作業員・職長が利用する「作業員向けアプリ（Worker App）」の2つのインターフェースを提供し、日々の作業日報の入力から、プロジェクトの進捗度・損益予測の把握までを一貫して行います。

## 2. このシステムの思想
**「日々の作業実績からリアルタイムに現場の損益を把握し、赤字リスクを未然に防ぐ」**
従来の月末締めや工事完了後にしか正確な原価が分からないというどんぶり勘定体制から脱却し、作業ごとの目標工数（ターゲット）に対して今どれだけの工数を消費しているかを日次でトラッキングします。
現場を指揮する「職長」自身が現在の損益状況と進捗をスマホで直感的に確認できるようにすることで、現場レベルでのコスト意識を向上させ、適切な軌道修正を可能にすることを目的としています。

## 3. このシステムの詳細な使用方法

### 管理者側の使用方法（管理アプリ / Admin App）
1. **ホーム**
   現在進行中・予定・完了のすべての現場の全体進捗と予測損益を一覧で俯瞰します。
2. **システム設定・作業員管理**
   全体で適用する標準の「時間単価（または人工）」を設定します。また、作業員の登録や「職長」の権限設定を行います。
3. **工事設定**
   新しい現場データを作成し、見積書（Excel形式など）を取り込むことで「作業項目ごと」の「目標工数」と「想定金額」をシステムへ自動生成します。
4. **管理シート（進捗・損益管理）**
   各作業項目の「実績トータル工数」「現在の進捗率（％）」から、「予測最終工数」を算出し、作業項目ごとの予測損益状態を自動計算によってリアルタイム表示します。
5. **実績出力**
   現場従業員が入力した実績を集計し、作業員単位での日報のエクスポートなどを行います。

### 現場側の使用方法（作業員向けアプリ / Worker App）
1. **作業実績の入力（一般作業員）**
   各自のスマートフォンからログインし、自身の参加している現場と本日の作業項目を選択します。日々の通常稼働時間と残業時間を入力して実績を提出します。
2. **現場進捗の更新と確認（職長）**
   職長権限を持つユーザーは「職長画面」にアクセスし、担当現場における各作業項目の「進捗率（％）」を更新・申告します。
   同時に画面上で「目標工数に対する消費割合」と「現在の予測損益」をリアルタイムで確認でき、現場を指揮しながら採算状況を常に把握・管理します。

## 4. このシステムの将来的な追加機能

今後のシステムの拡張として、蓄積されたデータをフル活用して原価計算のさらなる高精度化を実現する、以下の追加機能を計画しています。

### １．見積書作成システム
これまで担当者の経験則によって決定されていた作業項目ごとの単価を改善します。
本システムを運用することで蓄積されていくデータ（類似作業における実際の消費工数や標準的な作業スピードなど）を解析し、より現実的で根拠のある単価をシステムが自動で導き出します。これにより、実態に即した赤字になりにくい高精度な見積書を作成することが可能となります。

### ２．材料管理システム
現在のシステムでは、主におおよその「1人工あたりの平均費用（労務費）」に基づくことで現場の損益計算を行っています。今後はこの仕組みを拡張し「材料費」も統合管理する機能を導入します。
労務費だけでなく各作業項目において使用した材料・部材のコストも精密に追跡可能とすることで、より詳細な作業項目ごとの原価実績と単価設定を実現でき、現場の利益管理の解像度が飛躍的に向上します。
